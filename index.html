BERLIN EMPLOYEE ANALYTICS

Brutalist employee performance dashboard with cold precision.

Project Information

This is an employee analytics dashboard built with Python and Streamlit.

To run locally:

  1. Clone this repository
  2. Install dependencies with pip install -r requirements.txt
  3. Run streamlit run src/app.py
  4. Visit the provided localhost URL

GitHub Pages cannot host the full functionality of this application. Please run it locally for the complete experience.
